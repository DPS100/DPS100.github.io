Work Experience

I worked at DeNOVO Solutions for the past two summers as a software engineering intern on two separate projects.

My first role was helping an agile team of software engineers develop a full-stack Atlassian Jira plugin to track finances and work progress. This included fetching and writing user data to an H2 SQL database, serving dynamically generated webpages using a Java servlet engine, building RESTful request endpoints, designing good-looking and responsive pages using JavaScript, HTML, and CSS, and testing/integrating new deployments on an AWS EC2 instance.

My second role was prototyping and building a ML application from the ground up on an agile team of three interns. The goal of the app was to correlate separate geospatial data streams accurately with the fewest amount of false positives in a noisy environment. I contributed by writing bash scripts to automate deployments and creating docker-compose files to configure the application as a microservice architecture. Some of these microservices included setting up interfaces for a Kafka message broker and web sockets to communicate between services and streaming live data, an ELK (Elasticsearch, logstash, kibana) stack to store and display data, custom pipelines to save and retrieve data from the NoSQL elasticsearch database, and the core correlation service. I created the core correlation service, which was a custom ML unsupervised clustering algorithm that would correlate disparate spatiotemporal data streams. My contributions helped the microservices cold start from completely shut down to fully operational in under a minute, and being able to correlate tens of thousands of data points in mere minutes, a task that would take data analysts days to compare one by one.
